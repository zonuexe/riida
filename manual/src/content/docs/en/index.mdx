## Key Features

- **Library management** — Auto-index PDFs and EPUBs from multiple folders, tracking file changes
- **Full-featured viewer** — PDF viewer with text selection, links, and reading position memory
- **Tags and metadata** — Tag books and edit titles, authors, covers, and more
- **Notes** — Per-book floating notes with autosave
- **External books** — Register Kindle purchases and other file-less books in your library
- **Themes** — Four atmospheric themes to match your reading mood
