riida is a desktop reading app and library manager for your local PDF and EPUB collection.

It was built for readers who own hundreds or thousands of ebooks and want a single bookshelf with a built-in reader, fast search, notes, and metadata management.

## Key Features

### Library Management

- Auto-index PDFs and EPUBs from your chosen folders
- Real-time tracking of file additions, changes, and deletions
- Multiple library root folders supported
- Exclude patterns (glob format) to filter what gets scanned

### Reading

- Built-in PDF viewer with text selection, links, and full-text search
- Reading position saved and restored per file automatically
- Binding direction auto-detection (works with Japanese vertical-text PDFs)
- EPUB viewer (in development)

### Organizing Your Books

- Tag-based classification
- Edit metadata: title, authors, publisher, language, cover image, and more
- Bulk tag and metadata editing for multiple books
- Shelves for custom curated collections

### Notes and External Books

- Floating per-book notes (Milkdown editor, autosaved)
- Register Kindle purchases and other file-less books as virtual library entries

## Design Philosophy

riida is built around a "reading-first" principle:

- Features should not interrupt the reading experience
- Warm, editorial visual language — closer to a reading room than a productivity dashboard
- A tool that feels at home on a desktop, not in a browser tab

## Platform Support

riida currently targets macOS as its primary platform. Windows and Linux builds are provided, but receive less testing than the macOS version.

## Name

The name "riida" comes from "Reader."
