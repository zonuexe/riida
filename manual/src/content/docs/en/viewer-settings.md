Viewer settings let you customize the reading experience per-file or globally.

## Setting Scopes

| Scope | Description |
|-------|-------------|
| **Global** | Default settings applied to all books |
| **This file** | Settings for the current book only (overrides global) |

Switch between scopes using the tabs in the Viewer Settings panel. The file scope tab has a stronger visual tint to distinguish it.

## PDF Settings

### Page Layout

- **Single page**: one page at a time
- **Spread (left-to-right)**: for Western books
- **Spread (right-to-left)**: for Japanese books
- **Auto-detect**: inferred from the PDF content (default)

### Viewer Background Color

Choose the background color of the reading surface:

- Default / Snow White / Night City / Navy Blue
- Inherit app setting (default)

## EPUB Settings

### Font Size

Adjust the font size used for reading.

### Background Color

Choose the reading surface background color and matching text color. The same options as PDF are available.

## Resetting File Settings

To remove file-specific settings and fall back to global defaults, click **Reset file settings** in the file scope tab.
