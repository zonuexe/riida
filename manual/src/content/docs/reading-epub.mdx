<Aside type="caution" title="開発中の機能">
EPUBビューアは現在開発中の機能です。基本的な読書には使えますが、リンクの動作など一部に既知の問題があります。
</Aside>

## 基本操作

### ページ操作

| 操作 | 方法 |
|------|------|
| 次のページ | <kbd>→</kbd> / <kbd>PageDown</kbd> |
| 前のページ | <kbd>←</kbd> / <kbd>PageUp</kbd> |

### 読書位置

ビューアを閉じると現在の位置（<abbr title="Canonical Fragment Identifier">CFI</abbr>）が自動保存され、次回開いたときに前回の位置から再開します。

## ビューア設定

ツールバーの設定アイコンからフォントサイズ、背景色、レイアウトを変更できます。

## 既知の問題

### リンクの動作

EPUBコンテンツ内のリンクは現時点では正常に動作しない場合があります。

- 外部リンク（`https://`）: クリックしても反応しないことがある
- 同一セクション内アンクリンク（`#anchor`）: クリックするとページ表示が崩れることがある

この問題はmacOSのWKWebView環境でのクロスフレームDOM操作に起因しており、調査・対応中です。

### その他の制限

- 複雑なCSS・フォント指定を持つEPUBではレイアウトが崩れることがある
- 画像が多いEPUBでは読み込みに時間がかかることがある
