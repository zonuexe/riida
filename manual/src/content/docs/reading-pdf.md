riidaのPDFビューアはPDF.jsをベースにしており、テキスト選択・リンク・全文検索・読書位置の自動保存に対応しています。

## 基本操作

### ページ操作

| 操作 | 方法 |
|------|------|
| 次のページ | <kbd>→</kbd> / <kbd>PageDown</kbd> / スクロール |
| 前のページ | <kbd>←</kbd> / <kbd>PageUp</kbd> |
| ページ指定 | ツールバーのページ番号欄に入力 |
| 先頭ページ | <kbd>Home</kbd> |
| 末尾ページ | <kbd>End</kbd> |

### ナビゲーション

- **戻る / 進む**: ツールバーの ← / → ボタン、または <kbd>⌘[</kbd> / <kbd>⌘]</kbd>（macOS）
- **ページ内リンク・外部リンク**: クリックで自動的に移動

## テキスト選択とコピー

テキストレイヤーが含まれるPDFでは、ドラッグでテキストを選択してコピーできます。

スキャンされたPDF（テキストレイヤーなし）ではテキスト選択はできません。

## PDF内検索

ツールバーの検索アイコン（または <kbd>⌘F</kbd>）で検索バーを開きます。ヒットした箇所はハイライト表示されます。

## 読書位置の自動保存

ビューアを閉じたり別の書籍に切り替えたりすると、現在のページと位置が自動保存されます。次回開いたときに前回の位置から再開します。

## 見開き表示

ビューア設定からページレイアウトを変更できます。

- **単ページ表示**: 1ページずつ表示
- **見開き表示（左綴じ）**: 左から右へ読む書籍向け
- **見開き表示（右綴じ）**: 右から左へ読む日本語書籍向け
- **自動検出**: PDFの内容から綴じ方向を自動判定（デフォルト）

日本語縦書きPDFは自動検出で正しく右綴じ表示されます。

## ビューア設定

ツールバーの設定アイコンからPDF表示の詳細設定を変更できます。

設定には **グローバル**（全書籍共通）と **このファイル**（ファイルごと）の2つのスコープがあります。ファイルスコープの設定はグローバル設定を上書きします。

## ノート

PDFを開いた状態でツールバーのノートアイコンをクリックすると、フローティングノートパネルが開きます。内容は自動保存されます。
